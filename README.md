# ATIVIDADE2_LOPAL


# Exercícios de Lógica de Programação (LOPAL)

Este repositório contém uma coleção de algoritmos fundamentais desenvolvidos em **Pseudocódigo (Visualg)** e suas respectivas representações visuais em **Fluxogramas (Draw.io)**.

## Autor
* **Enzo Ferrari**
* **Data:** Abril de 2026

---

## Padronização dos Fluxogramas

Para facilitar a leitura, todos os diagramas seguem esta estrutura:
* **Início/Fim:** Elipses/Ovais.
* **Entrada de Dados:** Paralelogramos.
* **Processamento:** Retângulos.
* **Saída de Dados:** Trapézios.

---

## Tecnologias Utilizadas
* **Visualg - Para escrita e teste do código.
* **Draw.io** - Para criação da lógica visual.

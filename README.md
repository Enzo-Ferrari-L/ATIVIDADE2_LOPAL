## Padronização dos Fluxogramas

Para facilitar a leitura, todos os diagramas seguem esta estrutura:
* **Início/Fim:** Elipses/Ovais.
* **Entrada de Dados:** Paralelogramos.
* **Processamento:** Retângulos.
* **Saída de Dados:** Trapézios.

## Tecnologias Utilizadas
* **Visualg - Para escrita e teste do código.
